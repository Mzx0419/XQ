# 私域运营需求确认（总览 + 价值版）

## 一、功能总览

- 入口页面：`私域运营（VTfvR）`、`私域运营详情页（DEb7B）`
- 核心链路：筛选识别顾客 -> 进入详情诊断问题 -> 一键提醒触达 -> 任务/到店转化 -> 留存提升

### 1）总览页（私域运营）

- 按顾客维度筛选：徽章类型、任务类型、成长值、等级
- 快速识别高风险/高潜顾客，确定优先跟进对象
- 支撑管理者进行“分层运营”，减少无效触达

### 2）详情页（私域运营详情）

- 成长值总览：等级、进度、累计获得/累计兑换
- 任务信息：查看顾客偏好任务类型，支持分享链接直达任务
- 优惠券信息：识别临期券，推动预约到店
- 任务明细：查看近期完成任务与成长值变化，支撑复盘
- 关键动作：通过“去提醒”分享微信小程序/链接给顾客

## 二、价值定位

- 目标：顾客留存活跃度提升
- 活跃判定原则：任务与到店都计入活跃，但到店权重更高

### 1）核心业务价值

- 识别价值：从“凭经验运营”升级为“可筛选、可分层、可定位”
- 触达价值：从“发现问题”到“立即提醒”闭环更短
- 转化价值：通过任务参与与到店双驱动提升活跃与复访
- 经营价值：通过临期券跟进提升预约率与到店几率

### 2）关注指标（建议）

- 7日活跃顾客率（加权，且到店权重高于任务）
- 7日到店预约率
- 临期券转预约率
