# 医美仪器用次登记 - 门店沟通稿

## 要做什么

服务顾客时用了哪台仪器、用了多少次数/片数，要**扫仪器上的码 + 拍两次照（或填两次数字）**记下来，方便后面查和统计。

---

## 谁来做、怎么做

**库管**

- 仪器**入库**时：像现在一样走入库，系统会给这台仪器一个**唯一二维码**（贴机器上）。
- 仪器**返厂维修**：做出库，选「返厂维修」；修好回来再入库，**还是用原来那个码**。
- 机器彻底不能用了：打**报废**，不用换码。

**技师**

1. 先开单，在**领用**里找到这笔服务订单，点**扫码**。
2. **服务前**：扫仪器上的二维码 → 拍仪器屏幕（或直接填「开始前」的数字）。
3. 做完服务后，**结束时**：再扫一次同一台仪器的二维码 → 拍屏幕或填「结束后」的数字。
4. 系统自己算：用了多少 = 结束后 − 开始前。如果结束后比开始前还小，会**报错**，不让通过。

**注意**：订单取消了，已经记过的仪器用次**不删**，因为确实用过了。
